# ğŸ“Œ Overview
**ReuniteIt** is a full-stack **Lost & Found Management System** built using **Node.js, Express, MongoDB, EJS**, and **Cloudinary**.

Users can:
- Report **Lost** items  
- Report **Found** items  
- Search items  
- Manage all reports  
- Edit profile  
- View item details  

Admin can:
- Approve / Reject reports  
- Delete users and reports  
- View analytics dashboard  

This project includes a **modern UI**, SEO meta tags, responsive design, and Cloudinary image uploads.
